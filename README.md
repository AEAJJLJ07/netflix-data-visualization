# Анализ данных и классификация контента Netflix (Недели 1–3)

Репозиторий содержит финальный объединенный проект по комплексному анализу данных (EDA) платформы Netflix и обучению модели машинного обучения для предсказания типа контента.

## 📋 Структура проекта
* **data/raw/** — исходный датасет `netflix_titles.csv` (8807 строк).
* **notebooks/** — рабочие Jupyter-ноутбуки (`01_eda.ipynb`, `02_visualization.ipynb`) и итоговый автоматизированный скрипт `eda_netflix.py`.
* **img/img/** — сгенерированные графики аналитики.

---

## 🛠 Выполненные этапы

### 1. Неделя 1–2: Разведочный анализ (EDA) и Визуализация
Разработан скрипт для автоматического анализа структуры данных, обработки пропусков и генерации ключевых инсайтов. Созданы и сохранены следующие графики:

* **Распределение пропусков:** Визуализация незаполненных данных по колонкам (директория, актерский состав, страна).
  
  ![Распределение пропусков](missing_values.png)

* **Соотношение типов контента:** Круговая диаграмма, демонстрирующая долю Фильмов (Movies) и Сериалов (TV Shows).
  
  ![Соотношение типов контента](type_distribution.png)

* **Топ-10 жанров:** Горизонтальный столбчатый график самых популярных категорий на платформе.
  
  ![Топ-10 жанров](top_genres_horizontal.png)

### 2. Неделя 3: Машинное обучение (ML)
Решена задача бинарной классификации текстового контента:
* **Предобработка:** Текстовые признаки (`listed_in` + `description`) объединены и векторизованы с помощью метода **TF-IDF** (выделено 3000 ключевых фичей).
* **Разделение выборки:** Данные разбиты на обучающую (80%) и тестовую (20%) выборки со стратификацией.
* **Моделирование:** Обучен классификатор **Random Forest Classifier** (`n_estimators=100`).
